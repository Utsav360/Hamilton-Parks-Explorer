# Hamilton-Parks-Explorer

Hamilton Parks Explorer is a web-based application that allows to search for Parks in the City of Hamilton and display best driving routes to them by embedding Google Maps, Geocoding API, Geolocation API and using VanillaJS, Boostrap, JSON, CSS3, and HTML5
